1.2.1 位置矢量

首先确定参考系和坐标系，原点为 $O$。

质点 $P$ 可用 $x$，$y$，$z$ 三个坐标确定，记位矢

$$\overrightarrow{OP} = \mathbf r = x\mathbf i + y\mathbf j + z\mathbf k$$

位矢的大小

$$\left|\mathbf r\right| = r = \sqrt{x^2 + y^2 + z^2}$$

位矢的方向余弦

$$\cos\alpha =\frac x r,\ \cos\beta =\frac y r,\ \cos\gamma =\frac z r$$

运动方程

$$x=x(t),\ y=y(t),\ z=z(t)$$

或

$$\mathbf r = \mathbf r (t)$$

知道了运动方程，就能确定任意时刻质点的位置，从而确定质点的运动。 力学的基本任务之一，正是根据各种问题的具体条件，求解质点的运动方程。

质点在空间的运动路径称为轨迹。质点的运动轨迹为直线时，称为直线运动。质点的运动轨迹为曲线时，称为曲线运动。

从运动方程中消去 $t$ 可得轨迹方程。

轨迹方程和运动方程最明显的区别在于，轨迹方程不是时间 $t$ 的显函数。
